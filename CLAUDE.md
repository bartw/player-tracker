## Agent skills

### Issue tracker

Issues live as GitHub Issues on `bartw/player-tracker` (uses the `gh` CLI). PRs are not a triage surface. See `docs/agents/issue-tracker.md`.

### Triage labels

Canonical roles map 1:1 to label names (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`, `bug`, `enhancement`). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context: one `CONTEXT.md` + `docs/adr/` at the repo root. See `docs/agents/domain.md`.
